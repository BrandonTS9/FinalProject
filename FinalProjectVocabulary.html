Term Definition
var x = Will determine the value or contents of variable x.
x += 4; Adds 4 to the value of x each iteration of code.
for(var x = 4; x < 7; x++){} As long as the variable x has a value less than 7, a value of 1 will be added to x which has an initial value of 4.
if(x == 0){}

ellipse(_,_,_,_); Will draw an ellipse on the screen.
rect(_,_,_,_); Will draw a rectangle on the screen.
stroke(_,_,_);
fill(_,_,_);		 Will determine the color of the line or shape.
mouseX
mouseY		 Global variables that are used to record the position of the cursor on the screen.

Computer An electronic device that can store information according to instructions.
Input A hardware device that sends data to a computer.
Output A hardware device that displays or sends data from a computer.
Motherboard The central circuit board that connects all essential hardware for a computer.
CPU Interprets programs and processes data.
RAM Allows for quick and temporary storage, otherwise known as random access memory.
Power Supply Supplies electrical energy to a computer.
Hard Disk Drive Large and permanent secondary storage for a computer.
Solid State Drive Same purpose as a hard disk drive, but faster and smaller.
Analog Stored as analogies and found in the real world. This type of data is infinite.
Digital Stored as binary bits and organized in data bases. This type of data is quantized, unlike analog data.
Sampling The process of taking analog data and "compressing" it into digital data.
Binary Base 2 system used by computers. 0, 1, 2, 4, 8, 16 etc.
Algorithm A series of logical steps followed in order to complete a task.
List Has a fixed order, but new elements can be added.
Array Fixed number of elements, but the elements can switch places.
Linear Search A search algorithm that searches a data set in a straight line.
Adding in Binary Step 1: Line up the numbers.
Step 2: Calculate the new number knowing that 1 + 0 = 1, 0 + 0 = 0, and 1 + 1 = 0. Start from the right and head left.
Processing Time The amount of time that it takes for an algorithm to complete all of its steps.
Binary Search A search algorithm that selects the middle number in an ordered data set and eliminates half the list depending on whether the target number is less than or greater than the middle number.
Bubble Sort Goes through a list as many times as necessary, swapping each pair of elements in the incorrect order. The largest element is guaranteed to be ordered at the end of the first pass.
Merge Sort This is a divide and conquer sorting algorithm. First, the data set is divided into two seperate lists until there are only groups of two or one elements. Then, these groups are sorted and placed back together as a whole data set.
Selection Sort Each pass, the lowest element is selected and moved to one side of the list. This is repeated until the data set is sorted.
Insertion Sort First, one random element from an unsorted list is declared sorted. Another element from the unsorted list is chosen and placed properly into the sorted list. This continues until the data set is sorted.
Quick Sort This is also a divide and conquer sorting algorithm. First, a random pivot element is chosen. Second, the rest of the elements in the data set are moved to the left of to the right of the pivot according to their value. Then, in both of the new lists on the left in right, a new pivot for each list is chosen and the process is repeated until the data is sorted.
Greedy Algorithm Chooses the least valued element each pass. For example, for finding the shortest route on a road trip with multiple stops, a greedy algorithm would find the shortest distance by selecting the locations with the shortest distances at all given opprotunities.
Internal Links Links embedded in a webpage that transfer the user to another section of the page.
External Links Links embedded in a webpage that transfer the user to another webpage.
HTML Stands for Hyper Text Markup Language and is used to create websites.
CSS Stands for Cascading Style Sheets and is used to style and decorate HTML code.
<style> A HTML tag that begins the area where CSS code can be inserted.
<h1> and <h6> HTML tags that begin headers where text can be inserted. <h1> is the largest header and <h6> is the smallest header.
<strong> and <em> HTML tags that can alter the style of text. <strong> will bold the text and <em> will slant the text.
Storyboard This is a rough sketch of an idea. For a website storyboard, the sketch would include rough layout of the website, basically a general idea of what the final website will be.
<div> This is a HTML tag that is used to encompass a images, text, tables, etc. A class or id assigned to the <div> will allow the surrounded items to be styled with CSS.
OSI Model Stands for Open Systems Interconnection model. This was created by a multinational organization with the purpose of allowing network connections through the internet.
Application Layer Begins in the host computer where data is packed and prepared to be sent.
Presentation Layer Translates the data from the application layer to a format that can be read throughout the other layers.
Session Layer Allows the sender and recipient(s) to communicate to increase security.
Transport Layer Scans the data to make sure it is safe and valid.
Network Layer The data is separated into packets and the path the data will take is determined. Also adds the IP address.
Data Link Layer Adds the hexidecimal MAC address to avoid collisions with other data.
Physical Layer The data is encoded into a physical form to be sent through the internet.
